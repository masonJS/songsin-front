# songsin-front

songsin university project based SPA to Vue.js

## Structure

src
  - api
  - assets
  - components
  - routes
  - store
  - utils
  - views
  

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### deploy
```
npm run deploy
```
